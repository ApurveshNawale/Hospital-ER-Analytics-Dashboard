# 🏥 Hospital ER Analytics Dashboard | Power BI Project

## Project Summary
Built an interactive **Hospital Emergency Room (ER) Analytics Dashboard** using **Power BI** to analyze **9,216 patient records (19 months)** and uncover insights on patient flow, wait times, and operational efficiency.

Focus: Transforming raw healthcare data into **actionable business insights**.

---

## Business Problem
Hospitals struggle with:
- Long patient wait times  
- Poor resource allocation  
- Lack of visibility into peak demand  
- Fluctuating patient satisfaction  

➡️ This dashboard enables **data-driven decision-making** for hospital operations.

---

## 📊 Key Impact
- Identified **peak hours (11 AM & 11 PM)** → optimized staffing decisions  
- Found **average wait time = 35 mins** (above 30-min target)  
- Only **62% patients served within target time**  
- Highlighted **high-load departments (General Practice, Orthopedics)**  
- Improved visibility into **patient flow & bottlenecks**
